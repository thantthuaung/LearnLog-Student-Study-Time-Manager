## Key Implementation Details

### BaseFragment Logic
```kotlin
if (isTopLevelDestination) {
    // Show hamburger, open drawer
    button.setImageResource(R.drawable.ic_menu)
    button.onClick { activity.openDrawer() }
} else {
    // Show back arrow, navigate up
    button.setImageResource(R.drawable.ic_arrow_back)
    button.onClick { navController.navigateUp() }
}
```

### MainActivity Drawer Lock
```kotlin
navController.addOnDestinationChangedListener { _, destination, _ ->
    when (destination.id) {
        R.id.settingsFragment -> {
            // Lock drawer so swipe doesn't open it
            drawerLayout.setDrawerLockMode(LOCK_MODE_LOCKED_CLOSED)
        }
        else -> {
            // Unlock for normal pages
            drawerLayout.setDrawerLockMode(LOCK_MODE_UNLOCKED)
        }
    }
}
```

### Profile Binding (MainActivity)
```kotlin
lifecycleScope.launch {
    userPreferences.displayName.collect { name ->
        displayNameTextView.text = name.ifEmpty { "Add your name" }
    }
}
// Similar for email and avatar
```
