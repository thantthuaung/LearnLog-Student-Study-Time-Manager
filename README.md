# LearnLog - Student Study Time Manager

A comprehensive Android study tracking app with timer, task management, planner, and analytics.

## Quick Start

### Build the App
```bash
./gradlew assembleDebug
```

### Features
- ✅ **Tasks**: Create and manage study tasks
- ✅ **Timer**: Focus timer with Pomodoro support
- ✅ **Planner**: Schedule study sessions
- ✅ **Insights**: Study analytics and statistics
- ✅ **Notes**: Quick note-taking
- ✅ **Analytics**: Detailed performance analysis

## Tech Stack
- **Language**: Kotlin
- **Architecture**: MVVM
- **DI**: Hilt
- **Database**: Room
- **UI**: Material 3, Jetpack Compose
- **Charts**: MPAndroidChart
- **Navigation**: Jetpack Navigation

## Requirements
- Android Studio Hedgehog or later
- Min SDK: 26 (Android 8.0)
- Target SDK: 34 (Android 14)
